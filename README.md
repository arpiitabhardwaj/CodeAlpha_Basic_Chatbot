🤖 Smart ChatBot

About the Project

This is a simple Smart ChatBot developed using Python as part of my CodeAlpha Python Internship. The chatbot can interact with users, respond to basic questions, display the current date and time, and provide a friendly chatting experience.

This project helped me practice Python fundamentals such as functions, loops, conditional statements, and user input handling.

---

Features

- Greets the user personally
- Responds to common questions
- Shows current date and time
- Includes a help command
- Allows users to exit the chat easily
- Interactive and beginner-friendly

---

Technologies Used

- Python
- Datetime Module

---

What I Learned

While building this project, I learned:

- How to use functions in Python
- How loops work in real applications
- How to handle user input
- How to use conditional statements effectively
- How to work with Python modules like datetime

---

How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Run the following command:

python chatbot.py
